# Commissioning Workspace
Commissioning Workspace is a cross-platform desktop application for planning, executing, and documenting industrial commissioning works. It centralizes the management of projects, equipment assets, checklists, test records, punch lists, progress, and status.

## Requirements
- Node.js
- npm
- Rust
- Tauri system dependencies

## Run
- Install dependencies: `npm install`
- Start the application: `npm run tauri dev`
